AI Abalone Game

This 3D project is a digital version of the classic two-player board game Abalone, built in Godot. It features an AI opponent that uses the Minimax algorithm with alpha-beta pruning, a custom evaluation function, forward pruning with the Search Beam method, and table transposition for efficient state caching. The AI makes strategic decisions and can play full matches against itself, showcasing deep search and advanced pruning techniques in a real-time game environment.

Role: Lead Programmer, AI Engineer
Tech Used: Godot, GDScript
